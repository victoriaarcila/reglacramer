Regla de Cramer

A cada matriz cuadrada le podemos asignar un determinante, y este se representa de la siguiente forma:

Determinantes para una matriz de orden 2x2:
Estas tienen la siguiente forma: 

            a11    a12
            a21    a22

Reglas para resolver una matriz de orden 2x2:
El determinante de una matriz 2x2 es el resultado de la resta de los productos de sus diagonales. Es decir: 

            a11    a12
            a21    a22

            Determinante = (a11 * a22) - (a21 * a12)

Ejemplo: Dada la siguiente matriz, calcular su determinante: 

            B = 
            -7    8
            9   -6

            Determinante = (-7 * -6) - (9 * 8)
                        = 42 - 72
                        = -30

Ejemplo 2: Dada la siguiente matriz, calcular su determinante: 

            126    186
            23     93

            Determinante = (126 * 93) - (23 * 186)
                        = 11718 - 4278
                        = 7440

Para matrices de orden 3x3: Aplicando la regla de Sarrus
Una matriz de orden 3x3 tiene la siguiente forma: 

            a11    a12    a13
            a21    a22    a23
            a31    a32    a33

Reglas para resolver:
Copiamos la matriz original y debajo de ella copiamos exactamente los elementos de la fila 1 y la fila 2.

Los elementos de la diagonal de color azul se multiplican y se suman aplicando la ley de signos, si es el caso. Las otras diagonales, de color negro, se multiplican, se resuelven las operaciones y luego se restan los resultados obtenidos.

Ejemplo: Calcular el determinante de la siguiente matriz aplicando la ley de Sarrus: 

            a11    a12    a13
            a21    a22    a23
            a31    a32    a33

Para esta matriz, los cálculos serían: 

            |  2   4   5 |
            |  6   7   3 |
            |  3   0   2 |  : 28

            |  2   4   5 | 
            |  6   7   3 | 
            |  2   4   5 |  : 0

            |  6   7   3 | 
            |  6   7   3 | 
            |  6   7   3 |  : 36

Otros resultados de las multiplicaciones: 

            5.7.3 = 105

            3.0.2 = 0

            2.4.6 = 48

Finalmente, el determinante de la matriz sería: 

            |A| = 28 + 0 + 36 - 105 + 0 + 48
            |A| = 64 - 153
            |A| = [-97]
